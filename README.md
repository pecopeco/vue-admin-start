# vue-admin-start

> vue admin 管理后台模板，使用 vue-element-admin 框架搭建，删除多余插件和配置并调整项目结构，添加一些个人习惯使用的组件，包含 Element UI、fly、pug、stylus等

## Build Setup

# 安装依赖
npm install

# 启动服务
npm run dev
```

浏览器访问 [http://localhost:9528](http://localhost:9528)

## 发布

```bash
# 构建测试环境
npm run build:stage

# 构建生产环境
npm run build:prod
```
